URL Structure
Path-based is recommended. Query-based is also supported for backward compat.
Path-based (recommended) 
# /api/watch/{source}/{id}/{ep}/{type}?server=Name
# id = AniList ID  OR  mal-{malId}

GET /api/watch/senshi/20/1/sub
GET /api/watch/dao/mal-21/5/dub
GET /api/watch/wave/1535/12/sub?server=Vidstreaming
GET /api/watch/animeheaven/{heavenId}/3/sub
GET /api/watch/miruro/20/1/sub?server=bonk-sub
Query-based (compat) 
GET /api/watch?source=senshi&anilistId=20&ep=1&type=sub
GET /api/watch?source=dao&malId=21&ep=5&type=dub&server=Megacloud
Endpoints
Click to expand params and example responses.
GET /api/search Search anime by title via AniList ▼
ParamTypeRequiredDescription
qstringrequiredSearch query
Example Response 
{
  "query": "naruto",
  "count": 5,
  "results": [{
    "id": 20,          // ← use this as anilistId
    "malId": 20,
    "title": "Naruto",
    "coverImage": "https://s4.anilist.co/...",
    "episodes": 220,
    "status": "FINISHED",
    "format": "TV"
  }]
}
GET /api/info Anime metadata + all site-specific IDs ▼
ParamTypeRequiredDescription
anilistIdnumberone ofAniList ID
malIdnumberone ofMyAnimeList ID
Example Response 
{
  "anilistId": 20,
  "malId": 20,
  "title": "Naruto",
  "siteIds": {
    "zoro": "naruto-225",       // used by senshi, dao, wave
    "gogoanime": "naruto",
    "animeheaven": "ukr6y"
  }
}
GET /api/episodes Full episode list from a source ▼
ParamTypeRequiredDescription
anilistId / malIdnumberone requiredAnime identifier
sourcestringoptionalsenshi · dao · wave · animeheaven · miruro (default: senshi)
Example Response 
{
  "anilistId": 20, "source": "senshi", "count": 220,
  "episodes": [
    { "num": 1, "id": "ep-id-001", "title": "Enter! Naruto Uzumaki!" }
  ]
}
GET /api/servers All available servers for an episode ▼
ParamTypeRequiredDescription
anilistId / malIdnumberone requiredAnime identifier
epnumberrequiredEpisode number
typestringoptionalsub · dub · raw · all (default: sub)
sourcestringoptionalsenshi · dao · wave · miruro (default: senshi)
Example Response 
{
  "episode": 1, "type": "sub", "source": "senshi",
  "servers": [
    { "name": "Megacloud",    "sourceId": "abc123", "type": "sub" },
    { "name": "Vidstreaming", "sourceId": "def456", "type": "sub" }
  ]
}
GET /api/watch/{source}/{id}/{ep}/{type} Embed URL + m3u8 + subtitles ▼
ParamTypeWhereDescription
sourcestringpathsenshi · dao · wave · animeheaven · miruro
idstringpathAniList ID or "mal-{malId}"
epnumberpathEpisode number
typestringpathsub · dub · raw
serverstringqueryPreferred server name (optional, e.g. "Megacloud")
Example Response 
{
  "anilistId": 20, "malId": 20, "title": "Naruto",
  "episode": 1, "type": "sub", "source": "senshi",
  "server": "Megacloud",
  "availableServers": ["Megacloud", "Vidstreaming"],
  "embedUrl": "https://megacloud.blog/embed-2/e-1/AbCdEf",
  "m3u8": "https://cdn.megacloud.blog/hls/abc/master.m3u8",
  "subtitles": [{ "lang": "English", "url": "https://.../en.vtt", "default": true }],
  "intro": { "start": 0, "end": 90 },
  "outro": { "start": 1350, "end": 1440 },
  "note": null    // non-null = m3u8 failed, use embedUrl instead
}
